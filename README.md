# Phishing to Ransomware Incident Response Simulation Lab

This project demonstrates a controlled cybersecurity incident simulation where a phishing attack leads to a ransomware-style payload execution.

# Tools Used
- GoPhish
- Suricata IDS
- Sysmon
- Wireshark
- Postfix & Dovecot

# Lab Architecture
- Kali Linux (Attacker)
- Debian (Mail + IDS Server)
- Windows 10 (Victim)

# Learning Objectives
- Understand phishing attack vectors
- Analyze ransomware behavior
- Collect forensic evidence
- Reconstruct incident timelines
